# JOBRA-Projectt
Project Description

The JOBORA project is a Database Management System (DBMS)-based job portal designed to connect job seekers with employers efficiently. The system manages user data, job postings, applications, and hiring processes through a structured relational database.

The database is designed using multiple interconnected tables such as Users, JobSeekers, Employers, Jobs, Applications, and JobCategories. Each table is normalized and linked using primary and foreign keys to maintain data integrity and reduce redundancy.

The platform allows job seekers to register, add their skills, and apply for jobs, while employers can post job openings and manage applications. The system includes important functionalities implemented using SQL such as stored procedures for applying to jobs, updating application status, and searching jobs based on keywords, location, and skills.

Additionally, a view (JobApplicationStatus) is created to provide a combined and simplified overview of job applications, including job title, applicant name, and application status. This improves data accessibility and reporting.
